# 宝塔面板自定义字体插件

> 支持上传自定义字体文件并更改面板前端显示字体，旨在便于用户提高页面可读性，增强面板个性化
## 功能特性

- ✅ **字体上传** - 支持上传 TTF 格式字体文件
- ✅ **常规/粗体设置** - 可分别设置常规字体和粗体字体  
- ✅ **全局替换** - 全局替换面板所有文字显示
- ✅ **实时预览** - 提供字体预览功能，支持切换常规/粗体预览
- ✅ **一键恢复** - 支持一键恢复默认字体
- ✅ **版本管理** - 支持字体版本管理和重命名

## 安装方法

1. 下载插件压缩包到你的电脑
2. 在 宝塔面板 - 软件商店 - 第三方应用 - 导入插件 内进行上传
3. 在宝塔面板软件商店 - 第三方应用 - 中找到此插件
4. 安装完成

## 注意事项

- 请用户注意上传字体的是否允许商用以及版权所有，若引发纠纷本人概不负责
- 仅支持 TTF 格式字体文件
- 字体文件最大支持 10MB
- 建议使用支持中文的字体文件
- 更换字体后需要刷新页面生效
- 面板更新后可能需要重新应用字体设置

## 常见问题

**Q: 为什么部分字体没有生效？**
A: 请确保刷新浏览器页面，或清除浏览器缓存后重试。

**Q: 如何恢复默认字体？**
A: 点击"概览"页面的"恢复默认字体"按钮。

**Q: 支持哪些字体格式？**
A: 目前仅支持 TTF 格式，确保最佳兼容性。
